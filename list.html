⚙️ Setup Required (one-time, ~3 min)

  1. Go to jsonbin.io → Sign up free
  2. Click + Create Bin → paste {"items":[]} → Save
  3. Copy the BIN ID from the URL bar
  4. Go to API Keys → create a Secret Key (your Master Key)
  5. Create an Access Key → select your bin → Read-Only
  6. Open this HTML file in a text editor → fill in the CONFIG block at the top
  7. Upload to GitHub Pages (instructions below the list)

🌐 FREE HOSTING (GitHub Pages):

  1. Go to github.com → Sign up / log in
  2. New repo → name it anything → set to Public → Create
  3. Upload this list.html file → rename it index.html
  4. Settings → Pages → Branch: main → folder: / (root) → Save
  5. Your URL: https://yourusername.github.io/reponame — ready in ~1 min
